# 🏡 ML-Based Buyer Segmentation & Investment Profiling for Real Estate

## 📌 Overview

This project uses **Machine Learning (K-Means Clustering)** to segment real estate buyers into distinct groups based on their demographics, investment behavior, and financial patterns.

The goal is to help real estate businesses move from **generic marketing → data-driven targeted strategies**.

---

## 🚀 Features

* Buyer segmentation using **K-Means clustering**
* Identification of **4 key buyer personas**
* Interactive **Streamlit dashboard**
* Data-driven insights on:

  * Investment behavior
  * Geographic distribution
  * Financing patterns
  * Customer satisfaction

---

## 🧠 Buyer Segments

The model identifies 4 major segments:

* **Global Investors** → High satisfaction, diversified portfolios
* **First-Time Buyers** → Low satisfaction, need support
* **Corporate Buyers** → High loan usage, premium investments
* **Luxury Investors** → High-value, experienced investors

## 📂 Dataset

The dataset includes:

* 2,000 client records
* 10,000 property transactions

### Key Features:

* Demographics (age, gender, region, country)
* Investment metrics (total investment, avg price, property count)
* Behavioral attributes (loan usage, purpose, satisfaction)

---

## 📈 Methodology

1. Data Cleaning & Preprocessing
2. Feature Engineering
3. Encoding & Scaling
4. K-Means Clustering
5. Cluster Evaluation (Elbow + Silhouette)
6. Segment Interpretation

---

## 📊 Results
